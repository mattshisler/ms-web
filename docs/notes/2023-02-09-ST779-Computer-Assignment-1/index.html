visits1 <- rep(0, num.states)

for (i in 1:num.steps){

  visits1[current.state] <- visits1[current.state] + 1
  current.state <- sample(1:num.states, 1, prob = P[current.state,])

Step 8

Write your own conclusions based on this simulation experiment and give an explanation of the phenomenon you observe.

When simulating \(n=1000\) steps of the Markov chain the initial state seems to influence the proportion of visits to each state. The reason we observe this phenomenon is that the chain has not “burned in” to the stationary distribution and run sufficiently long for the initial transitions to be obscured in the calculated proportions.

We obtain the a good estimate of the stationary distribution when taking the transition matrix to a high power, however. This is different than simulating the chain from an intial state. Taking powers of the transition matrix computes transition probabilities from any state to any other state in the alloted number of steps, a much richer operation than observing one sample path in the simulation approach.
